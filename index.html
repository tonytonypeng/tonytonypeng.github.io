python 系统编程学习

Posted on 2017-08-29

系统编程包括有进程和线程，相关代码可阅读github

1、进程

fork
  • os.fork()，子进程返回0，父进程返回子进程的ID。window下不支持fork。父子进程执行没有顺序。
  • getpid获取当前进程ID，getppid获取当前进程的父进程ID。
  • 孤儿进程：就是父进程先结束，子进程会由1号进程进行回收。

  • 僵尸进程：子进程先结束，而父进程一直不结束对子进程进行回收。

  • 每个进程之间的资源是独立的，不能修改，包括全局变量。

    Process

  • multiprocessing(跨平台)模块提供了一个Process类来代表一个进程对象。

  • Process类常用属性：

  • 回收子进程：多进程且死循环

Pool
  • 进程池：p = Pool()
Queue

  • 进程间通信：q = Queue()

  • 进程池间通信

2、线程

  • t = threading.Thread(target=xxx，args=(xxx))

  • 线程的run方法

  • 共享全局资源：当多个线程同时调用全局资源时会造成混乱。
  • 同步：解决资源共享问题。就是按照约定，当一个线程执行的时候，另一个线程不执行。

  • 互斥锁：哪里修改全局资源就在哪里加锁。

  • 死锁：拥有各自锁的线程都想拿到对方的锁。

  • 银行家算法：通过提前的计算(某种算法)的形式来分配资源。

  • 生产者和消费者模式：由于生产能力和消费能力不一致，需要使用阻塞队列降低耦合。

  • 异步：一方发送数据，在没有得到另一方会回应前可以继续执行下去。

Python 高级语法学习总结

Posted on 2017-08-29

高级语法包括有迭代器、生成器、面向对象进阶等，由于内容较多，这里不做长篇累述，相关代码可阅读我的github

1、高级语法

  • 迭代器：iter()返回一个迭代器，next()函数遍历可迭代对象。适合已存储数据的对象。
  • 生成器：列表生成器的中括号改为括号或包含yield关键字。适合动态生成数据的对象。
  • 闭包：闭包：内部函数和外部函数的数据。修改外部变量可用nolocal声明、
    装饰器

  • 语法糖@+闭包函数。

  • 被装饰的函数带有(不定长)参数，def inner[(*arg,*kwarg)参数]。

  • 装饰器带有参数,再加一层def。
  • 类装饰器:__call__ ，当把对象当做函数使用时会自动调用。
  • functools：包含一些工具函数。

2、细节

  • import 导入模块

  • 作用域

  • =，is：=判断内容是否一样，is判断变量引用的地址是否相同。
  • 深拷贝、浅拷贝：深浅是对于数据传递的顺序。越接近原始数据越浅。

3、面向对象进阶

  • python是动态语言：可在运行中给类、对象添加或删除属性和方法。
  • __slots__：可以设置给实例添加的属性范围
  • 私有化：_xx(私有);__xx(与子类区分);__xx__(魔法方法);x_(与python关键字区分)
  • 属性property：一类装饰器，通过私有属性方法进行装饰，外部可以进行访问和赋值。
  • 元类:万物皆对象。元类就是类的类，type。

4、垃圾回收

  • 手动回收：gc.collect()

5、其他

  • 内建属性
  • 内建函数
  • 集合set

6、调试

  • 待定

7、编码风格

  • 待定

python 基础语法学习

Posted on 2017-08-26

基础语法主要包括循环判断语句、文件操作、函数和面向对象的学习。github的基础语法学习代码.

一、循环判断语句总结

  • if往往用来对条件是否满足进行判断。
  • while循环一般通过数值是否满足来确定循环的条件
  • for循环一般是对能保存多个数据的变量，进行遍历

二、函数总结

  • 函数是由def+函数名(形参)+(文档说明，调用help查看)+代码构建而成，它的作用主要为了减少重复代码。

  • 引用：在python中，值是靠引用来传递来的。我们可以用id()来判断两个变量是否为同一个值的引用。

  • 递归函数：自己调用自己，例如处理阶乘。
  • 匿名函数：lambda [arg1 [,arg2,…..argn]]:expression

三、文件操作

  • 打开、关闭、读、写文件
  • 定位读写：tell、seek(offset,from)
  • os 模块下的文件操作

四、面向对象（封装、继承、多态）

  • 类和对象：类是创建对象的模板。类：类名(大驼峰)、属性、方法。
  • __init__初始化属性，__str__打印对象(return)，__del__删除对象。
  • 属性：实例属性和类属性。

  • 方法：一般方法：当重写了父类方法，如还需父类的功能，手动调用。还包括：类方法(\@classmethod)和静态方法(classmethod)

  • 继承：单继承和多继承。私有属性不会继承。

  • 多态：让父类变量能引用子类对象。
  • 几种设计模式：单例模式和工厂模式
简单的面向对象应用
  • 飞机大战

五、异常

  • 1、try–>except–>else:没有异常时执行–>finally：有无异常都会执行。

    • 不指定

    • 指定

    • 多个异常

    • 默认

  • 2、异常传递：被调用的函数出现异常没有处理会传递给调用者。

  • 3、自定义异常：抛出自定义异常再捕获异常。

  • 4、异常中抛出异常：在except中raise异常。

六、模块

  • 一个单独的py文件为一个模块。

  • __name__:值为__main__时本地测试用；值为模块名时被导入可用。

七、其他

  • 包(文件夹)： from…import…
  • 定位搜索：sys.__file
  • 接收终端参数：sys.argv，即终端接收的参数
  • 列表推导式:[x for x in range()]
  • set、list、tuple：三者可以相互转换，set为元素不重复的{}。
