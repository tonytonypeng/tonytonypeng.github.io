CSS

  • css(层叠样式表)：行内式、嵌入式（内联式）、外链式：
  • 字体：font-family：汉字/英文/unicode编码。
  • 字体大小：font-size，px为像素。
  • 字体颜色：color。
  • 字体加粗：font-weight:bold
  • 字体斜体：font-style:italic

  • 字体的连写：字体大小和行高必须在字体类型之前。

  • 下划线：text-decoration:underline(增加)/none(删除)。

  • 文本位置：text-align:left/center/right
  • 首行缩进：text-indent：为px时需要知道字体大小，默认16px/为em时设为当前文字大小的倍数。
  • 行高：line-height，文字高度+上下间距。
  • 边框：border:solid实线/dash虚线/dotted点线。

  • 外边距：margin:上、右、下、左（顺时针）

  • 内边距：padding，特性与外边距一样。

  • 选择器：标签选择器、类选择器、层级选择器、ID选择器、并集选择器、伪类选择器。
  • 浮动：float,只有左和右。
  • 元素溢出：overflow：visiable(默认显示多余的内容)/hidden(隐藏)/scroll(滚动查看)/auto(自动)。
  • ID选择器：#id名{}，id是唯一的。
  • 并集选择器/组选择器：a,b{}。

定位
  • 相对定位：relative，以自己为参照物，即使走了，还占位置。
  • 绝对定位：absolute，以父元素为准(父元素必须定位(相对)，脱离文档流，不占位置。
  • 固定定位：fixed，以浏览器为准，脱离文档流，不占位置。
  • 无定位：static。
  • 定位层级：z-index，值越大，越靠前。
  • 设置圆角：border-tadius。
表格

  • 跨列合并：colspan，找到第几行的第几个。
  • 跨行合并：rowspan，找到第几列的第几个。

其他选择器
  • 其他选择器权重都为1。
  • 通用选择器:*{属性:值}。
  • E>F{属性:值};子元素选择器,匹配所有属于E元素的子元素F。
  • E+F{属性:值}; 相邻元素选择器,匹配所有紧随E元素之后的元素(下面)。

  • 属性选择器:标签[属性class] 针对所有是class属性的元素。

  • E:nth-child(n)：匹配元素类型为E且是父元素的第n个子元素。

  • E:nth-of-type(n) 第几个E类型元素。

transition动画
  • transition-property 设置过渡的属性，比如：width height background-color
  • transition-duration 设置过渡的时间，比如：1s 500ms
  • transition-timing-function 设置过渡的运动方式，有 linear(匀速)/ease(缓冲运动)
  • transition-delay 设置动画的延迟
  • transition: property duration timing-function delay 同时设置四个属性
transform
  • transform 形变属性：rotate旋转、scale缩放、translate平移、skew斜切。
  • translate(x,y) 设置盒子位移
  • scale(x,y) 设置盒子缩放
  • rotate(deg) 设置盒子旋转
  • skew(x-angle,y-angle) 设置盒子斜切
  • perspective 设置透视距离
  • transform-style flat | preserve-3d 设置盒子是否按3d空间显示
  • translateX、translateY、translateZ 设置三维移动
  • rotateX、rotateY、rotateZ 设置三维旋转
  • scaleX、scaleY、scaleZ 设置三维缩放
  • tranform-origin设置变形的中心点,接受两个参数，它们可以是百分比，em，px等具体的值，也可以是left,center,right，或者 top,center,bottom等描述性参数 ;
  • backface-visibility 设置盒子背面是否可见
