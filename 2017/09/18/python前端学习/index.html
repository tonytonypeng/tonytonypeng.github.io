CSS

  • css(层叠样式表)：行内式、嵌入式（内联式）、外链式：
  • 字体：font-family：汉字/英文/unicode编码。
  • 字体大小：font-size，px为像素。
  • 字体颜色：color。
  • 字体加粗：font-weight:bold
  • 字体斜体：font-style:italic

  • 字体的连写：字体大小和行高必须在字体类型之前。

  • 下划线：text-decoration:underline(增加)/none(删除)。

  • 文本位置：text-align:left/center/right
  • 首行缩进：text-indent：为px时需要知道字体大小，默认16px/为em时设为当前文字大小的倍数。
  • 行高：line-height，文字高度+上下间距。
  • 边框：border:solid实线/dash虚线/dotted点线。

  • 外边距：margin:上、右、下、左（顺时针）

  • 内边距：padding，特性与外边距一样。

  • 选择器：标签选择器、类选择器、层级选择器、ID选择器、并集选择器、伪类选择器。
  • 浮动：float,只有左和右。
  • 元素溢出：overflow：visiable(默认显示多余的内容)/hidden(隐藏)/scroll(滚动查看)/auto(自动)。
  • ID选择器：#id名{}，id是唯一的。
  • 并集选择器/组选择器：a,b{}。

  • 伪类选择器

  • 列表标签：

      为有序列表、
        为无序列表，
      • 中写文本。去掉前面的点：list-style：none。

  • 定义列表：dl、dt(标题)、dd(内容)
表单
  • form标签：属性：action(地址)、method(方法：get、post)。
  • label标签：标注、显示信息。

  • input标签：type为输入类型、name为表单key、value为值。

  • 下拉列表标签：select类型，option内容

  • margin外边距合并：以外边距较大的一方为准。
  • margin-top塌陷：父元素的边界未设置，子元素找了浏览器边界作为基准。
