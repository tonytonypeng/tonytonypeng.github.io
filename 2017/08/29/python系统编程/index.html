投稿日 2017-08-29

系统编程包括有进程和线程，相关代码可阅读github

1、进程

fork
  • os.fork()，子进程返回0，父进程返回子进程的ID。window下不支持fork。父子进程执行没有顺序。
  • getpid获取当前进程ID，getppid获取当前进程的父进程ID。
  • 孤儿进程：就是父进程先结束，子进程会由1号进程进行回收。

  • 僵尸进程：子进程先结束，而父进程一直不结束对子进程进行回收。

  • 每个进程之间的资源是独立的，不能修改，包括全局变量。

2、线程

  • t = threading.Thread(target=xxx，args=(xxx))

  • 线程的run方法

  • 共享全局资源：当多个线程同时调用全局资源时会造成混乱。
  • 同步：解决资源共享问题。就是按照约定，当一个线程执行的时候，另一个线程不执行。

  • 互斥锁：哪里修改全局资源就在哪里加锁。

  • 死锁：拥有各自锁的线程都想拿到对方的锁。

  • 银行家算法：通过提前的计算(某种算法)的形式来分配资源。

  • 生产者和消费者模式：由于生产能力和消费能力不一致，需要使用阻塞队列降低耦合。

  • 异步：一方发送数据，在没有得到另一方会回应前可以继续执行下去。
