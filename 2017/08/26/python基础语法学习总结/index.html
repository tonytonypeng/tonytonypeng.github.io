python 基础语法学习

Veröffentlicht am 2017-08-26

基础语法主要包括循环判断语句、文件操作、函数和面向对象的学习。github的基础语法学习代码.

循环判断语句总结

  • if往往用来对条件是否满足进行判断。
  • while循环一般通过数值是否满足来确定循环的条件
  • for循环一般是对能保存多个数据的变量，进行遍历

函数总结

  • 函数是由def+函数名(形参)+(文档说明，调用help查看)+代码构建而成，它的作用主要为了减少重复代码。

  • 引用：在python中，值是靠引用来传递来的。我们可以用id()来判断两个变量是否为同一个值的引用。

  • 递归函数：自己调用自己，例如处理阶乘。
  • 匿名函数：lambda [arg1 [,arg2,…..argn]]:expression
