python 基础语法学习

Posted on 2017-08-26

基础语法主要包括循环判断语句、文件操作、函数和面向对象的学习。github的基础语法学习代码.

一、循环判断语句总结

  • if往往用来对条件是否满足进行判断。
  • while循环一般通过数值是否满足来确定循环的条件
  • for循环一般是对能保存多个数据的变量，进行遍历

二、函数总结

  • 函数是由def+函数名(形参)+(文档说明，调用help查看)+代码构建而成，它的作用主要为了减少重复代码。

  • 引用：在python中，值是靠引用来传递来的。我们可以用id()来判断两个变量是否为同一个值的引用。

  • 递归函数：自己调用自己，例如处理阶乘。
  • 匿名函数：lambda [arg1 [,arg2,…..argn]]:expression

三、文件操作

  • 打开、关闭、读、写文件
  • 定位读写：tell、seek(offset,from)
  • os 模块下的文件操作

四、面向对象（封装、继承、多态）

  • 类和对象：类是创建对象的模板。类：类名(大驼峰)、属性、方法。
  • init初始化属性，str打印对象(return)，del删除对象。
  • 属性：实例属性和类属性。

  • 方法：一般方法：当重写了父类方法，如还需父类的功能，手动调用。还包括：类方法(@classmethod)和静态方法(classmethod)

  • 继承：单继承和多继承。私有属性不会继承。

  • 多态：让父类变量能引用子类对象。
  • 几种设计模式：单例模式和工厂模式

五、异常

  • 1、try–>except–>else:没有异常时执行–>finally：有无异常都会执行。

    • 不指定

    • 指定

    • 多个异常

    • 默认

  • 2、异常传递：被调用的函数出现异常没有处理会传递给调用者。

  • 3、自定义异常：抛出自定义异常再捕获异常。

  • 4、异常中抛出异常：在except中raise异常。

六、模块

  • 一个单独的py文件为一个模块。

  • name:值为_main时本地测试用；值为模块名时被导入可用。

七、其他

  • 包(文件夹)： from…import…
  • 定位搜索：sys.__file

  • 接收终端参数：sys.argv，即终端接收的参数

  • 列表推导式:[x for x in range()]

  • set、list、tuple：三者可以相互转换，set为元素不重复的{}。
