IP

  • ip地址：标记主机在网络中的位置，包括网络地址和主机地址，且是唯一的。

  • 私有ip

  • IP地址127.0.0.1 ~ 127.255.255.255用于回路测试。

  • 子网掩码：与ip地址相与可得出网络号。
端口
  • 网络中用来标识进程id，0~65535。
  • 知名端口：0~1023；80：HTTP；21：FTP。
  • 动态端口：1024~65535。
  • 查看端口：netstat -an。

  • tcp长连接:避免了频繁套接字的建立和销毁socket，会长时间占用资源。一般用于视频传输、文件传输、频繁请求等业务。
